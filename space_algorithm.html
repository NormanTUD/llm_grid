Step 1 — Tokens in Embedding Space

Input text is tokenized and each token gets an initial embedding vector. We also add diverse "probe" tokens for better spatial coverage. Here they are projected to 2D via PCA.

Token embeddings (Layer 0)

Step 2 — Residual Stream Deltas

For each token at each layer, we compute how much the layer "pushes" the token:

δtoken,layer = hlayer+1[token] − hlayer[token]

Each arrow below shows the delta vector for one token at the selected layer.

Token positions + delta arrows

Step 3 — RBF Interpolation (The Key Algorithm)

We lay a regular grid over the space. For each grid vertex, we compute a weighted average of all known token deltas using Gaussian RBF kernels:

wi = exp( −‖pgrid − ptoken_i‖² / (2σ²) )

δgrid = Σ wi · δtoken_i / Σ wi

This extends the discrete per-token deformations into a continuous vector field across the entire space. Hover over the grid to see the RBF weights from that point.

Interpolated velocity field on grid
RBF weight heatmap (hover on left canvas)

Step 4 — Grid Deformation

Each grid vertex is displaced by the interpolated velocity, scaled by parameter t and amplification amp:

p'grid = pgrid + t × amp × δgrid

Drag the t slider to watch the space morph in real time!

Deformed grid + token trajectories

Step 5 — Strain Visualization

Strain measures how much the space is stretched or compressed between adjacent grid points:

strain = ‖p'j − p'i‖ / ‖pj − pi‖

Strain > 1 → expansion (red)  |  Strain < 1 → contraction (blue)  |  Strain ≈ 1 → preserved (gray)
